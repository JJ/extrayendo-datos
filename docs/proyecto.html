Markdown es un lenguaje que permite escribir textos estructurados y con marcas de forma bastante simple. Nosotros lo usaremos en el README. md, para describir qué hace el programa y qué hace falta para instalarlo. En este punto quizás es conveniente elegir un verdadero editor de programadores tal como Atom. Atom, por ejemplo, te presenta directamente en dos paneles si lo configuras el fichero en Markdown y el resultado de convertirlo a HTML u otro formato, aparte de usar colores relacionados con la sintaxis.
